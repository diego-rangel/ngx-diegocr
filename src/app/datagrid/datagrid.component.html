Inputs

Name Type Description
tableId
Default: automatically generated 		string A custom id for the table inside the datagrid.
data any[] The data to show into the Datagrid.
columns DataGridColumnModel[] The columns definitions.
autoFitMode
Default: ByContent 		EnumAutoFitMode

Defines the way the datagrid should handle the columns sizes.

Default: All collumns will try to fit the entire available space. It's recommend when you have a small number of columns.

ByCaption: All collumns will respect the width that you provided into the columns width property. It's recommended when you want to define every single width for all your columns.

ByContent: This is a special mode, where the datagrid will automatically keep your columns fitting the necessary space by their content. It's the most dynamic mode, strongly recommended to use.
emptyResultsMessage
Default: No results found at this moment. 		string The message to show when the Datagrid's model is empty or null.
infoMessage
Default: {{'Showing records from {recordsFrom} to {recordsTo} of {totalRecords} records found.'}} 		string The info messages to show for representing the Datagrid's info.
animated
Default: true 		boolean Enables some awesome animations effects.
striped
Default: true 		boolean Applies a style for even & odd background-color.
bordered
Default: true 		boolean Applies a style to get a bordered grid.
hoverEffect
Default: true 		boolean Applies a style of effects on hover.
responsive
Default: true 		boolean Defines when the DataGrid should be responsive or not. It'll create an horizontal scroll when the page space is not enought.
initialColumnToSort number The column index that should be initially sorted.
initialSortDirection
Default: Ascending 		EnumSortDirection When initialColumnToSort is provided, the column sort direction that should be initially sorted.
mode
Default: OnClient 		EnumDataGridMode Defines if sorting and paging should be OnServer or OnClient.
totalItems number When mode is OnServer, you must provide the totalItems manually.
itemsPerPage
Default: 10 		number Maximum number of items per page. If value is less than 1, it will display all items in a single page.
maxSize
Default: 10 		number Limit number for page links in pager.
boundaryLinks
Default: true 		boolean If false, first and last buttons will be hidden.
directionLinks
Default: true 		boolean If false, previous and next buttons will be hidden.
rotate
Default: true 		boolean If true, current page will be in the middle of pages list.
showActionsColumn
Default: false 		boolean If true, a column for actions provided by ng-template will be displayed at the beginning of the table.
showCheckboxColumn
Default: false 		boolean When true, a column will be displayed at the beginning of the datagrid and each data object will have a selected property.
actionsColumnCaption string When showActionsColumn is true, the captions text of the actions column.
actionsColumnWidth
Default: 100px 		string When showActionsColumn is true, the width of the actions column.
allowExports
Default: false 		boolean If true, an export button will be visible for exportations.
exportButtonLabel
Default: Export 		string The export button's label.
exportedFileName
Default: Export 		string The exported file name.
exportedExcelSheetName
Default: Sheet 		string The exported Excel Sheet name.

Outputs

Name Type Description
OnPaginate EventEmitter<any> An event emitted when the Datadrig mode is defined to operate OnServer mode for handling paging operations. This callback will receive as a parameter an object with the current page number.
OnSort EventEmitter<DataGridColumnModel> An event emitted when the Datadrig mode is defined to operate OnServer mode for handling sorting operations. This callback will receive as a parameter an object with the sorted column.
OnRowSelected EventEmitter<any> An event emitted when showCheckboxColumn is defined to true and a row is selected/deselected. This callback will receive as a parameter an object with the checked row.
OnSelectionChanged EventEmitter<void> An event emitted when showCheckboxColumn is defined to true and any selection is changed.
