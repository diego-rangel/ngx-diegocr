LocalStorageService

Provides methods for localstorage management. Yout just need to inject the service wherever you want:

Functionalities:

Method Description
Get(key: string): any Returns the local stored value for the provided key.
Set(key: string, value: any): void Sets a value into the provided key.
Remove(key: string): void Removes the stored value for the provided key.

AuthService

Provides methods for JWT authentication management. Yout just need to inject the service wherever you want:

Functionalities:

Method Description
Autheticate(username: string, password: string): Observable<any> Sends an http POST for the configured enpoint into the ApiModule's settings and stores the access token into the local storage in case of success.
Logout(): void Removes the access token from the local storage.
IsAuthenticated(): boolean Checks if the access token keeps stored into the local storage.
GetToken(): string Returns the local stored access token.

ApiService

Provides methods for Api Http requests that encapsulates many Angular native complexities. All the following functionalities automatically handles JWT auth, including an "Authorization" header with the stored Access Token. Yout just need to inject the service wherever you want:

Functionalities:

Method Description
get(url: string, query?: KeyValuePair[], options?: ApiRequestOptions): Observable<any> Http GET request.
post(url: string, data: any, options?: ApiRequestOptions): Observable<any> Http POST request.
put(url: string, data: any, options?: ApiRequestOptions): Observable<any> Http PUT request.
delete(url: string, options?: ApiRequestOptions): Observable<any> Http DELETE request.

ApiRequestOptions object properties

Method Description
fullUrl When false, it will use the configured base path + url. Default: false.
headers Http custom Headers.
responseType ResponseContentType; Default: JSON.
autoMapResponse When true, it's going to automatically map the response based on the responseType.
