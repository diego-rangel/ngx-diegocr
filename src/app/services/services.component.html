HttpClientInterceptors

Usage

Interceptor Description
BaseApiUrlInterceptor Automatically add a base api url that you've provided into the apiSettings.
CommonHeadersInterceptor Automatically add common http headers like Content-Type, Cache-Control, Pragma and Expires.
JwtAuthInterceptor Automatically add an Authorization http header with a JWT according to your apiSettings.
WinAuthInterceptor Automatically set all your requests to be withCredentials.
ResponseDataInterceptor Modifies all response bodies according to the provided apiSettings.
ErrorHandlerInterceptor Provides a useful and unique contract for errors.

LocalStorageService

Provides methods for localstorage management. Yout just need to inject the service wherever you want:

Functionalities:

Method Description
Get(key: string): any Returns the local stored value for the provided key.
Set(key: string, value: any): void Sets a value into the provided key.
Remove(key: string): void Removes the stored value for the provided key.
