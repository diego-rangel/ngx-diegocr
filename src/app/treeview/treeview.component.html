Tree View

A 100% pure Angular+CSS Tree View implementation.
You can define as many columns as you need.

Inputs

Name Type Description
columns TreeViewColumnModel[] The columns definitions.
data any[] The data to show into the TreeView.
labelProperty string The property name for the value to show as the item's label.
collectionProperty string The property name for the collections.
emptyResultsMessage
Default: No results found at this moment. 		string The message to show when the TreeView's model is empty or null.
showIcons
Default: true 		boolean Defines if the icons should be visible or not.
normalItemIconClass
Default: fa fa-file 		string The normal icon class.
collapsableClosedItemIconClass
Default: fa fa-folder 		string The expand icon class.
collapsableOpennedItemIconClass
Default: fa fa-folder-open 		string The collapse icon class.
expandAllOnInit
Default: true 		boolean Defines if the TreeView should be initialized expanded or not.
enableLabelLinkStyle
Default: false 		boolean Defines if the TreeView's labels should be rendered as links or not.

Outputs

Name Type Description
OnItemExpanded EventEmitter<any> An event emitted when a row is expanded with the expanded data as parameter.
