ngx-ui-hero

An Angular v6 awesome library for easily build really nice apps based on bootstrap v4.

Powered by Diego Rangel Code on Github

Dependencies

You must install all the following dependencies in order to get all ngx-ui-hero functionalities working.

angular@^6.0.3 bootstrap@^4.1.1 file-saver@^1.3.8 font-awesome@^4.7.0 jquery@^3.3.1 lodash@^4.17.10 ng2-currency-mask@^5.3.1 ng2-file-upload@^1.3.0 ngx-bootstrap@^3.0.1 popper.js@^1.14.3 sweetalert2@^7.22.2 xlsx@^0.13.4

Installation

Modules

NgxUiHeroModule

Import this module to access the ngx-ui-hero Core functionalities.

 DataGrid new

NgxUiHeroDataGridModule

In order to use the ui-hero DataGrid, you must import this module with a bit of configurations.

 TreeView new

NgxUiHeroTreeViewModule

In order to use the ui-hero TreeView, you must import this module with a bit of configurations.

NgxUiHeroApiModule

In order to use the ui-hero Api functionalities like Base Http Services, Base Crud Services, Local Storage Services and JWT Authentication, you must import this module providing some configurations.

NgxUiHeroInputFormsModule

In order to use the ui-hero Forms functionalities like Input text, select, textarea, etc..., you must import this module providing some configurations.

In order to use the Input-date, you must include into your angular.json the following style "node_modules/ngx-bootstrap/datepicker/bs-datepicker.css".
