Tutorials

Usage

1) Inject the TutorialService in your appComponent or wherever you want.

2) Decorate as many elements you want with the tutorial directive.

3) Tell the TutorialService to PLAY!

Demo

TutorialService Methods

Method Description
addAction Used for programmatically add a tutorial into any element.
playAll It plays all tutorials that are being rendered with the tutorialDirective or added manually at that time.
playByKey It's similar to the playAll, but it'll play only the tutorials which has the provided key.
moveNext Programmatically moves to the next Tutorial.
movePrev Programmatically moves to the previous Tutorial.
exit Programmatically exits the Tutorial.

TutorialDirective Options

Property Type Description
key
opcional 		string An identification used when you want to show a certain group of tutorals by key.
title
opcional 		string The tutorial's block title.
text string The tutorial's block text.
order
opcional 		number Defines the ascendent order where each tutorial should be played.
