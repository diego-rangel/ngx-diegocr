Print HTML

It's only a simple wrapper for printThis.js, which makes our lives easier when printing.

The ReportComponent belongs to the NgxUiHeroModule Core module, so in order to use it, just be sure to import it.

In order to use the ReportComponent, you must include into your angular.json the following files:

  • "node_modules/ngx-ui-hero/print.css"
  • "node_modules/print-this/printThis.js"
